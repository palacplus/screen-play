

## Generate a new controller

dotnet-aspnet-codegenerator controller -name MoviesController -async -api -m Movie -dc ClimaxDbContext --relativeFolderPath Controllers


## Add a new database migration

dotnet ef migrations add [NameOfMigration] --context ClimaxDbContext


## Update database from migration

dotnet ef database update --context ClimaxDbContext
